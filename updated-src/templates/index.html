Offline Wallet with Dual Signatures

Use these endpoints:

  • POST /register {'username','password'} → returns public_id & embedded_id
  • POST /add_money {'username','amount'}
  • POST /login {'username','password'} → access_token
  • POST /generate_master {'amount'} (Authorization: Bearer token) → master token JSON
  • Run python3 scripts/sign_local.py master_token.json <sender_embedded_id>
    Enter receiver_embedded_id at prompt → produces signed_token.json
  • POST /redeem_token (Authorization: Bearer token) with signed JSON → redeem
